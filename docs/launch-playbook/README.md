## Guiding Principles

1. **Depth over breadth** — 50 researchers who use Lumen weekly > 5,000 who sign up and never return
2. **Earn trust, don't market** — Show up in communities with genuine value before mentioning Lumen
3. **Listen more than ship** — Beta is for learning, not for growing
4. **Founding Researchers are co-builders** — They shape the product, not just test it
5. **Product Hunt is a reward, not a starting line** — We launch publicly only after proving product-market fit with our beta cohort
