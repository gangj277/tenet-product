# Full error analysis for the training of deep neural networks

Christian Beck, Arnulf Jentzen, and Benno Kuckuck
ETH Zurich, University of Münster, University of Düsseldorf
January 31, 2020

This paper presents a comprehensive mathematical error analysis for a specific deep learning algorithm, decomposing the total error into approximation, generalization, and optimization errors. The authors estimate each of these components and combine them to provide an overall error analysis, establishing convergence with a convergence speed. While the established convergence speed is noted as suboptimal, slow, and subject to the curse of dimensionality, the paper's main contribution lies in its holistic approach to error analysis, covering all three major error sources and integrating them into a single estimate.

## Key Claims

*   The error in deep learning algorithms can be decomposed into approximation error, generalization error, and optimization error.
*   This work provides an error analysis for a specific deep learning algorithm by estimating each of these three error components.
*   The analysis establishes convergence with a convergence speed for the overall error of the considered algorithm.
*   The established convergence speed is far from optimal, is slow, increases exponentially with dimensions, and suffers from the curse of dimensionality.
*   The primary contribution is a full error analysis that covers all three error sources and merges them into one overall error estimate.

## Relevant Evidence/Quotes

*   "The error of a deep learning algorithm can in many situations be decomposed into three parts, the approximation error, the generalization error, and the optimization error."
*   "In this work we estimate for a certain deep learning algorithm each of these three errors and combine these three error estimates to obtain an overall error analysis for the deep learning algorithm under consideration."
*   "In particular, we thereby establish convergence with a suitable convergence speed for the overall error of the deep learning algorithm under consideration."
*   "Our convergence speed analysis is far from optimal and the convergence speed that we establish is rather slow, increases exponentially in the dimensions, and, in particular, suffers from the curse of dimensionality."
*   "The main contribution of this work is, instead, to provide a full error analysis (i) which covers each of the three different sources of errors usually emerging in deep learning algorithms and (ii) which merges these three sources of errors into one overall error estimate for the considered deep learning algorithm."

## Contribution to Research Frame

This source directly addresses the core concerns of the research frame by providing a *mathematical* breakdown of errors in deep learning. It explicitly identifies the three key error components (approximation, generalization, optimization) that are central to understanding performance degradation, which is analogous to the "model collapse" phenomenon in the context of synthetic data. The paper's finding that convergence speed is slow and suffers from the "curse of dimensionality" aligns with the research frame's concern about the risks of synthetic data loops and the potential for performance decay. While the source doesn't directly discuss synthetic data, its rigorous error decomposition provides a foundational mathematical perspective on why errors accumulate in complex models, which is crucial for understanding the theoretical underpinnings of model collapse. It supports the idea that errors can be systematically analyzed and potentially managed, even if the current analysis reveals limitations.
