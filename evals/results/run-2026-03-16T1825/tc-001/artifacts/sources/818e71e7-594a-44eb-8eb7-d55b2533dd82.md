# Asymptotic Analysis of Reliability in Recursively Defined Networks

**Author:** Tsitsiashvili G.Sh.
**Publication:** ProbStat Forum (e-journal)
**Date:** 2009

## Summary

This paper presents methods for accelerating the calculation of random network reliability, which is typically an exponentially complex problem. The author proposes using asymptotic formulas and analyzing recursively defined classes of random networks. Asymptotic formulas are derived for both low and high reliability scenarios, relating network reliability to minimal way lengths or cross-section abilities. The paper also introduces recursive formulas for calculating reliability and asymptotic parameters of networks constructed from generators, demonstrating that these calculations require a linear number of operations, a significant improvement over traditional methods.

## Key Claims

*   The exponential complexity of random network reliability calculations necessitates approaches that accelerate these computations.
*   Asymptotic formulas can approximate network reliability by assuming arcs are either very reliable or very unreliable, dependent on a small parameter $h$.
*   These asymptotic formulas connect network reliability to minimal way lengths (for low reliability) or minimal cross-section abilities (for high reliability).
*   Recursively defined classes of random networks, constructed by replacing arcs with previously defined networks, can be analyzed using recursive formulas.
*   The reliability calculations for recursively defined networks using these formulas require a linear number of arithmetic operations, offering a substantial efficiency gain.
*   The concepts of "reliability invariance" and "network narrow place" can be introduced through asymptotic analysis.

## Relevant Evidence/Quotes

*   "Problems of a calculation of a random network reliability has exponential complexity and demand a number of arithmetic operations which increases as a geometric progression by a number of a network arcs... So approaches which accelerate these calculations are actual."
*   "In this paper these approaches are based on asymptotic formulas and on a consideration of recursively defined classes of random networks."
*   "Recursive formulas are constructed to calculate reliabilities of random networks and their main asymptotic parameters. These formulas demand linear number of arithmetic operations."
*   Theorem 3.1 provides specific asymptotic formulas for network reliability ($P_\Gamma$) under different assumptions about arc reliability (e.g., $p_w = c_w h^{d_w}$ for low reliability, $p_w = c_w h^{s_w}$ for high reliability, and exponential forms).
*   Theorem 6.1 states that for recursively defined ports, the number of arithmetic operations for calculating reliability and its parameters is linear with respect to the number of arcs in the constructed network, contrasting with the geometric progression or higher-order polynomial complexity for general networks.
