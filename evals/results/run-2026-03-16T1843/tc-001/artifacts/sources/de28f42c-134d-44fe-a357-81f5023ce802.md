# Model Collapse Demystified: The Case of Regression

**Authors:** Elvis Dohmatob, Yunzhen Feng, Julia Kempe
**Year/Venue:** arXiv preprint arXiv:2402.07712v2 [cs.LG] 30 Apr 2024

## Summary

This paper provides a theoretical framework for understanding and mitigating "model collapse" in high-dimensional kernel regression trained on recursively generated synthetic data. The authors derive analytic formulae for test error, decomposing it into clean error, an increase in bias, and a multiplicative degradation term dependent on the number of training generations. They demonstrate that modified scaling laws exhibit new crossover phenomena and propose an adaptive regularization strategy to counteract collapse. The research highlights that optimal regularization for synthetic data differs from that for clean data and shows that model collapse can occur even without label noise, particularly in the over-parametrized regime.

## Key Claims

*   Model collapse in high-dimensional regression can be theoretically characterized using analytic formulae for test error under iterative retraining on synthesized data.
*   Test error under iterative retraining can be decomposed into clean test error, an increase in bias, and a multiplicative degradation term dependent on the number of generations.
*   Modified scaling laws for test error exhibit new crossover phenomena from fast to slow rates due to iterative synthetic data generation.
*   A simple strategy based on adaptive regularization can mitigate model collapse by using an optimal exponent that gracefully adapts to synthesized data.
*   In the absence of label noise, model collapse can manifest as an increase in bias, particularly in the over-parametrized regime, which can be an increasing function of the number of generations.
*   The optimal regularization parameter for mitigating model collapse differs from the optimal parameter for training on clean data, leading to novel crossover phenomena.
