# Full error analysis for the training of deep neural networks

**Publication Info:**
*   Authors: Christian Beck, Arnulf Jentzen, Benno Kuckuck
*   Date: January 31, 2020
*   Affiliations: ETH Zurich, University of Münster, University of Düsseldorf

## Summary

This paper presents a comprehensive mathematical error analysis for deep learning algorithms, addressing the lack of rigorous theoretical explanations for their success. The authors decompose the total error into three components: approximation error, generalization error, and optimization error. They then provide estimates for each of these errors for a specific deep learning algorithm and combine them into an overall error analysis, establishing convergence with a convergence speed. While the established speed is noted as suboptimal, slow, and suffering from the curse of dimensionality, the paper's main contribution is the full error analysis that covers and merges these three distinct error sources.

## Key Claims

*   The error in deep learning algorithms can be decomposed into approximation error, generalization error, and optimization error.
*   There is a lack of completely rigorous mathematical error and convergence analysis explaining the success of deep learning.
*   This work provides an estimation for each of the three error components for a specific deep learning algorithm.
*   These estimates are combined to form an overall error analysis, establishing convergence with a specific convergence speed.
*   The established convergence speed is suboptimal, slow, increases exponentially with dimensions, and is subject to the curse of dimensionality.
*   The primary contribution is a full error analysis that addresses all three error sources and merges them into a single estimate.

## Relevant Evidence/Quotes

*   "In this work we estimate for a certain deep learning algorithm each of these three errors and combine these three error estimates to obtain an overall error analysis for the deep learning algorithm under consideration."
*   "In particular, we thereby establish convergence with a suitable convergence speed for the overall error of the deep learning algorithm under consideration."

## Contribution to Research Frame

This source directly contributes to the research frame by providing a foundational mathematical framework for understanding errors in deep learning. It explicitly identifies and analyzes the three key error components (approximation, generalization, optimization) that are central to the problem of model collapse, which arises from accumulated approximation errors in recursive training. While the paper focuses on a specific algorithm and acknowledges limitations in its convergence speed analysis (suffering from the curse of dimensionality), its core contribution is the *methodology* for a "full error analysis" that merges these sources. This aligns with the research frame's goal of understanding the technical thresholds and data composition strategies that influence model collapse, by offering a theoretical lens through which to examine these phenomena. It supports the idea that errors accumulate and can be analyzed, which is a prerequisite for managing them.
