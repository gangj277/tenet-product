# Is Model Collapse Inevitable? Breaking the Curse of Recursion by Accumulating Real and Synthetic Data

**Publication:** arXiv e-prints
**Date:** April 2024
**DOI:** 10.48550/arXiv.2404.01413

## Summary

This paper investigates the phenomenon of "model collapse," where generative models trained on their own synthetic outputs progressively degrade in performance. Previous research suggested this was inevitable, assuming new data replaced old. However, this study explores a more realistic scenario where data accumulates. The authors empirically demonstrate that replacing real data with synthetic data leads to model collapse across various model types and data modalities (text, molecules, images). Crucially, they show that accumulating synthetic data alongside the original real data *avoids* model collapse. An analytical framework is used to prove that accumulating data results in a finite upper bound on test error, thus preventing collapse.

## Key Claims

*   Replacing original real data with successive generations of synthetic data leads to model collapse.
*   Accumulating synthetic data alongside the original real data avoids model collapse.
*   This effect holds across different model sizes, architectures, hyperparameters, and data modalities (text, molecule conformation, image generation).
*   An analytical framework proves that accumulating data bounds the test error, preventing indefinite degradation.

## Relevant Evidence/Quotes

*   "We confirm that replacing the original real data by each generation's synthetic data does indeed tend towards model collapse, then demonstrate that accumulating the successive generations of synthetic data alongside the original real data avoids model collapse."
*   "We obtain similar results for deep generative models on other types of real data: diffusion models for molecule conformation generation and variational autoencoders for image generation."
*   "we extend this argument to prove that if data instead accumulate, the test error has a finite upper bound independent of the number of iterations, meaning model collapse no longer occurs."

## Contribution to Research Frame

This source directly supports the inferred research frame by providing strong empirical and theoretical evidence that model collapse is *not* inevitable. It challenges the notion of collapse as a fundamental law by demonstrating a clear mitigation strategy: data accumulation. The findings suggest that the *way* synthetic data is integrated (replacing vs. accumulating) is a critical factor, aligning with the frame's emphasis on data composition strategies.
