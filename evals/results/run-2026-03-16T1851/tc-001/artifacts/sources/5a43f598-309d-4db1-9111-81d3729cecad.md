# Modeling Dependent Data: An Excursion

**Author:** Reinhard Furrer and the Applied Statistics Group
**Date:** Version June 20, 2023

## Summary

This document serves as lecture notes for a course on modeling dependent data, focusing on statistical concepts and methods for analyzing data that violates the independent and identically distributed (i.i.d.) assumption. It introduces various types of dependent data, including time series and spatial data, and explores different modeling approaches such as ARMA models, spectral methods, lattice models, and spatial processes. The notes also delve into Bayesian hierarchical modeling, Gibbs sampling, and Markov Chain Monte Carlo (MCMC) methods for estimation and simulation. The document aims to provide a comprehensive overview of techniques for handling complex data structures encountered in various scientific fields.

## Key Claims

*   The assumption of independent and identically distributed (i.i.d.) data is often violated in real-world scenarios, necessitating specialized modeling techniques.
*   Time series data exhibits temporal dependency, which can be modeled using concepts like autocovariance and models such as ARMA.
*   Spatial data, where observations are georeferenced, displays spatial dependency, requiring methods like conditionally autoregressive models, Gaussian Markov Random Fields, and geostatistical approaches (e.g., variogram, kriging).
*   Bayesian hierarchical models offer a powerful framework for complex dependency structures, often implemented using MCMC methods like Gibbs sampling and Metropolis-Hastings.
*   Different types of spatial data exist, including geostatistical data, lattice data, and spatial point patterns, each requiring specific modeling strategies.
